**Avaliação A1 – Técnicas de Desenvolvimento de Algoritmos**

**Sobre o Projeto**
Este projeto foi desenvolvido como parte da Avaliação A1 da disciplina Técnicas de Desenvolvimento de Algoritmos.
Ele apresenta um menu interativo contendo quatro funcionalidades principais que utilizam:

**Estruturas condicionais**

**Estruturas de repetição**

**Listas**

**Dicionários**

O programa é executado no terminal e permite interação com o usuário.

**Funcionalidades do Sistema**

Verificar etapa escolar (Condicionais)
O usuário informa sua idade e o sistema determina em qual etapa escolar ele pode se matricular:

Menor que 6 anos -> Não pode se matricular

Entre 6 e 14 -> Ensino Fundamental

Entre 15 e 17 -> Ensino Médio

18 anos ou mais -> EJA ou cursos para adultos

Mostrar pares de 1 a 100 (Repetição)
O programa utiliza duas estruturas de repetição: for e while.
Ele imprime todos os números pares entre 1 e 100.

**Cadastro de alunos (Listas)**
O usuário informa nomes de alunos que são armazenados em uma lista.
Para encerrar, digita "sair".
Ao final, o programa mostra:

A lista completa de alunos

A quantidade total cadastrada

**Cadastro de notas (Dicionários)**
O usuário informa o nome do aluno e sua nota.
As informações são armazenadas em um dicionário no formato:
aluno: nota
Para finalizar, o usuário digita "parar".
Ao final, o sistema exibe:

**Todas as notas cadastradas**

A média geral da turma

**Menu Principal**
1 - Verificar etapa escolar
2 - Mostrar pares de 1 a 100
3 - Cadastro de alunos
4 - Cadastro de notas
0 - Sair

**Como Executar o Programa**
Abra o arquivo .py em um editor Python.
Execute com o comando: python nome_do_arquivo.py
Escolha a opção desejada no menu.

**Tecnologias Utilizadas**

  Python 3
  
  Estruturas condicionais

  Estruturas de repetição (for e while)

  Listas

  Dicionários

  Funções

Objetivo Educacional
Demonstrar a compreensão dos conceitos básicos de lógica de programação utilizados no desenvolvimento de algoritmos.
